Comandos básicos

- git config
    Define as diretivas de configuração do Git
    Ex: Configuração do usuário e do e-mail para que o Git registre quem está fazendo as alterações no repositório
        e define o nome padrão da branch criada durante a inicialização para main (padrão GitHub).
        git config --global user.name "Marco Aurélio Biaggi"
        git config --global user.email "mafobiaggi@gmail.com"
        git config --global init.defaultBranch main

- git init
    Inicializa um repositório local vazio dentro do diretório selecionado
    Ex: Criação de um diretório local e inicialização de um repositório
        $ mkdir projeto-git
        $ git init
            Initialized empty Git repository in /home/suporte/Documents/Projetos/projeto-git/.git/

- git status
    Verifica se existe algum arquivo criado, modificado ou removido
    Ex:
        $ git status
            On branch main
            Your branch is up to date with 'origin/main'.

            Untracked files:
            (use "git add <file>..." to include in what will be committed)
                    Anotacoes/

            nothing added to commit but untracked files present (use "git add" to track)

    Observações:
        Arquivos com estado de Untracked são aqueles que não estão presentes no último commit, ou na última versão registrada.

- git add
    Adiciona arquivos para serem registrados no próximo commit. Ou seja, coloca eles no "Stage".
    Ex: Criação do arquivo comandos-basicos.txt e adição no stage.
        $ git add comandos-basicos.txt
            $ git status
            On branch main
            Your branch is up to date with 'origin/main'.

            Changes to be committed:
            (use "git restore --staged <file>..." to unstage)
                    new file:   comandos-basicos.txt

    Observações:
        Parâmetros:
            $ git add . (adiciona todos os arquivos novos na área de "Stage").
            $ git add -A (adiciona todas as modificações, inclusive exclusões de arquivos, deferente do comando outro comando).

- git commit 
    Registra ("commita") as modificações salvas em uma versão.
    Ex: Registra a criação do arquivo comandos-básicos.txt até esta linha.
        $ git commit -m "curso-git(add/Anotacoes): Criação da primeira versão do arquivo 'comandos-basicos.txt'."
            [main dde5056] curso-git(add/Anotacoes): Criação da primeira versão do arquivo 'comandos-basicos.txt'.
            1 file changed, 53 insertions(+)
            create mode 100644 Anotacoes/comandos-basicos.txt

    Observações:
        Parâmetros:
         $ git commit -m (adiciona uma mensagem a versão).
         $ git commit -a (mesma função do git add -A).
         $ git commit -am (simplificação dos comandos -a [--all] e -m [--message]).
         $ git commit --amend --no-edit (Faz alterações no commit passado sem alterar a mensagem. Usado para pequenas correções, sem a necessidade de um novo commit).

        Arquivos importantes:
            .gitignore: Não registra no commit os arquivos, ou extensões presentes no arquivo.
                Ex: .bak .~ => arquivos de backup e temporários respectivamente.
  
            .git-keep: Permite o registro de um diretório vazio na versão (commit).
                      Caso contrários, os diretórios vazios são ignorados por padrão

- git log
    Exibe todos as versões (commits) realizados no repositório
    Ex: Saída completa do log
        $ git log
            commit dcce929de4efc61e20a6d43c88c3072073d4d720 (HEAD -> main)
                Author: Marco Aurélio Biaggi <mafpbiaggi@gmail.com>
                Date:   Mon Nov 10 19:30:35 2025 -0300

                    curso-git(add): Criação do arquivo .gitignore.

                commit dde50564f999499ca33e246cb694ef65d5628836
                Author: Marco Aurélio Biaggi <mafpbiaggi@gmail.com>
                Date:   Mon Nov 10 19:12:13 2025 -0300

                    curso-git(add/Anotacoes): Criação da primeira versão do arquivo 'comandos-basicos.txt'.
        
        Saída simplificada do log (sem autor e data de execução)
            $ git log --oneline
                dcce929 (HEAD -> main) curso-git(add): Criação do arquivo .gitignore.
                dde5056 curso-git(add/Anotacoes): Criação da primeira versão do arquivo 'comandos-basicos.txt'.
